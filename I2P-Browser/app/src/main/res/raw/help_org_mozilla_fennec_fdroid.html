How to configure Fennec Firefox:

  1. Click HERE, and choose to open the URL in Fennec.
  2. Change the value of "network.proxy.type" to 1.
  3. Change the value of "network.proxy.http" to 127.0.0.1.
  4. Change the value of "network.proxy.http_port" to 4444.
  5. Change the value of "network.proxy.ssl" to 127.0.0.1.
  6. Change the value of "network.proxy.ssl_port" to 4444.
  7. Change the value of "media.peerconnection.ice.proxy_only" to true.
  8. You may also wish to install plugins to help manage your browser's default javascript settings, such as NoScript and/or uMatrix, and use Private Browsing to manage your browser's cache.
  9. You can now browse I2P eepsites!

Alternate Configuration(Requires Root!)

  1. Install user.js-updater
  2. Select Fennec from the list of browsers
  3. Paste the following URL where it says "type custom user.js url" https://github.com/eyedeekay/firefox.profile.i2p/raw/master/firefox.profile.i2p/user.js
  4. Tap "Update"
  5. You can now browse I2P eepsites!
